# Variables dynamiques pour le module How to Use

## Textes principaux
1. main_title : titre principal du module (ex: "Getting Started Guide")
2. main_subtitle : sous-titre descriptif (ex: "Follow these 3 easy steps to maximize your results.")

## Étape 1
3. step1_number : numéro de l'étape (ex: "1")
4. step1_title : titre de l'étape (ex: "Unbox Your Product")
5. step1_description : description de l'étape (ex: "Carefully remove your item from the packaging and check that all components are included.")
6. step1_image_url : URL de l'image de l'étape
7. step1_image_alt : texte alternatif de l'image (ex: "Step 1")

## Étape 2
8. step2_number : numéro de l'étape (ex: "2")
9. step2_title : titre de l'étape (ex: "Set Up in Minutes")
10. step2_description : description de l'étape (ex: "Follow the simple instructions to assemble and configure your product quickly and easily.")
11. step2_image_url : URL de l'image de l'étape
12. step2_image_alt : texte alternatif de l'image (ex: "Step 2")

## Étape 3
13. step3_number : numéro de l'étape (ex: "3")
14. step3_title : titre de l'étape (ex: "Experience the Difference")
15. step3_description : description de l'étape (ex: "Start using your product and enjoy the benefits of enhanced comfort and performance.")
16. step3_image_url : URL de l'image de l'étape
17. step3_image_alt : texte alternatif de l'image (ex: "Step 3")

## Couleurs principales
18. primary_color : couleur du titre principal (ex: "#1E5A8E")
19. badge_color : couleur de fond du badge numéroté (ex: "#3B82C7")
20. border_color : couleur de la bordure dashed des cartes (ex: "#7AB8E8")
21. gradient_start : début du gradient de l'image wrapper (ex: "#BDD9F2")
22. gradient_end : fin du gradient de l'image wrapper (ex: "#A3C9E8")

TOTAL : 22 variables (customisation complète sans surcharge)
