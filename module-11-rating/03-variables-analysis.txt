📋 NOTES IMPORTANTES
───────────────────────────────────────────────────────────────

✓ Module très compact et simple
✓ Pas de JavaScript nécessaire pour ce module
✓ Les étoiles sont statiques (pas interactives)
✓ Support responsive avec 3 breakpoints
✓ Système d'étoiles: filled_stars_count détermine combien 
  d'étoiles sont actives (couleur rose) vs inactives (gris)
✓ Badge avec background et texte personnalisables
✓ Compatible avec système de notation décimale (4.7, 4.5, etc.)

───────────────────────────────────────────────────────────────
🎨 UTILISATION DES VARIABLES
───────────────────────────────────────────────────────────────

ÉTOILES:
- Générer 5 SVG étoiles
- Les N premières (filled_stars_count) utilisent star_active_color
- Les restantes utilisent star_inactive_color

TEXTE NOTATION:
- Afficher rating_score_label en gras avec score_text_color
- Afficher (rating_count) en normal avec count_text_color

BADGE:
- Background: badge_bg_color
- Texte: badge_text_color
- Contenu: badge_text

───────────────────────────────────────────────────────────────
💡 VARIANTES POSSIBLES
───────────────────────────────────────────────────────────────

1. Nombre d'étoiles variable (3, 4, 5)
2. Demi-étoiles pour .5 (ex: 4.5 = 4 pleines + 1 demi)
3. Badge optionnel (peut être masqué)
4. Couleurs personnalisables (jaune, or, bleu, etc.)
5. Tailles d'étoiles variables (petit, moyen, grand)
