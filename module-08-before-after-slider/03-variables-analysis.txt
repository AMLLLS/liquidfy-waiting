NOTES IMPORTANTES :
- La variable ${initial_position} doit être utilisée à 2 endroits :
  1. Dans l'attribut value="" de l'input range
  2. Dans le style inline --position: XX%
- Les IDs doivent rester uniques si plusieurs sliders sur la même page
- Les images doivent avoir des ratios similaires pour éviter la déformation (object-fit: cover)
