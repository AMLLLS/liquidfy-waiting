───────────────────────────────────────────────────────────────
🎨 UTILISATION DES VARIABLES
───────────────────────────────────────────────────────────────

TEXTE D'ANNONCE:
- Afficher announcement_text en gras à gauche/centre
- Font-size: announcement_text_size (12px desktop, 10px mobile)
- Couleur: bar_text_color

BLOCS COUNTDOWN:
- 3 blocs minimum (heures, minutes, secondes)
- Optionnel: 4ème bloc pour les jours si show_days = true
- Chaque bloc contient:
  * <span> pour le chiffre (number_font_size, bold)
  * <small> pour le label (label_font_size, normal)

COULEURS:
- Barre: bar_bg_color + bar_text_color
- Blocs: block_bg_color + block_text_color + border

JAVASCRIPT:
- Calculer temps restant jusqu'à countdown_end_date
- Mettre à jour les spans #countdown-hours, #countdown-minutes, #countdown-seconds
- Si hide_when_expired = true → display:none quand temps écoulé
- Padding des chiffres: String(hours).padStart(2, '0')
